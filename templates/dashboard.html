System Architecture

This diagnostic support tool leverages state-of-the-art Convolutional Neural Networks (CNN) to analyze visual behavioral patterns. The core inference engine is powered by MobileNetV2, selected for its high efficiency and accuracy in extracting deep hierarchical features from complex visual media.

By processing these extracted features, the network identifies subtle visual markers correlated with specific behavioral metrics, providing quantifiable data to support clinical observation.

Dual-Model Routing

To ensure maximum accuracy, the system routes media to specialized, independent AI models based on the input format:

  • Static Image Engine: Trained exclusively on high-resolution spatial features from still photography.
  • Temporal Video Engine: A separate pipeline that extracts localized frames (1 FPS) to analyze behavioral changes and physical movements over time.

Live Telemetry

The Analysis Engine includes a secure, 5-second live webcam assessment tool. This feature captures real-time biometric telemetry, tracking variables such as:

  • Facial landmark stability and head pose estimation.
  • Eye Aspect Ratio (EAR) for involuntary blink rate detection.
  • Motor movement thresholds and localized repetitive motions.

⚠ Clinical Disclaimer

This application is a behavioral analysis prototype designed for educational, research, and technical demonstration purposes. It is not a certified medical device and must not be used to issue formal clinical diagnoses of Autism Spectrum Disorder (ASD). All outputs should be strictly reviewed by licensed medical professionals.
